# 🏁 Project Walkthrough: Excel Roadmap Chatbot Agent

ระบบเอเจนต์ผู้ช่วยติดตาม วิเคราะห์แผนงาน และจัดสรรงานอัจฉริยะจาก Google Sheets ทั้งสิ้น 7 ไฟล์ บน Google Drive สำหรับส่งประกวดผลงาน **Capstone Project** ได้รับการพัฒนา ติดตั้ง และรันการรันทดสอบอัตโนมัติสำเร็จครบถ้วน 100% เรียบร้อยแล้ว

---

## 🚀 1. โครงสร้างไฟล์และสถาปัตยกรรมที่พัฒนาเสร็จสิ้น

ไฟล์สำคัญทั้งหมดของโปรเจกต์ได้รับการจัดสร้างขึ้นดังนี้:
* **`pyproject.toml`** และ **`requirements.txt`**: สำหรับควบคุมจัดการ dependencies และการติดตั้งโครงการ
* **`mcp_server.py`**: ตัวเชื่อมต่อ Google Sheets API/Drive API (พร้อมระบบ Mock Fallback ปลุกการตอบกลับเมื่อทำงานออฟไลน์)
* **`app/agent.py`**: ตัวควบคุมประสานงาน (Multi-Agent Orchestrator) ที่วิเคราะห์หาความเสี่ยงโครงการและคิวว่างของนักพัฒนา
* **`app/security.py`**: ตัวกรองปิดบังข้อมูลส่วนบุคคล (PII Masking) ของเบอร์โทรศัพท์และอีเมล
* **`app/session_manager.py`**: ระบบจัดการประวัติการสนทนาและแคชไฟล์ (SQLite Cache Engine)
* **`main.py`**: หน้าต่างแดชบอร์ดปฏิสัมพันธ์ (Streamlit Portal) สำหรับผู้ใช้งาน
* **`.agents/skills/`**: โฟลเดอร์เก็บทักษะและ Instructions ของบอตอย่างเป็นสัดส่วน (4 ทักษะย่อย)
* **`tests/`**: ชุดการรันทดสอบ Pytest ครอบคลุมการวิเคราะห์ข้อมูลทั้ง 5 รูปแบบและฟีเจอร์ความปลอดภัย

## 🧪 2. ผลการรันทดสอบอัตโนมัติ (Automated Test Verification)

การรันคำสั่งทดสอบระบบ `python -m pytest` บนเครื่องผู้ใช้งานเสร็จสิ้นด้วยการ **ผ่านการทดสอบครบถ้วน 12/12 รายการ** อย่างสวยงาม:


### การตรวจสอบฟังก์ชันย่อยทั้ง 5 รูปแบบในชุดทดสอบ:
1. **งานรายบุคคล (`test_agent_task_tracking_pete`):** บอตค้นหาข้อมูลข้ามชีต ST และรายงานความเคลื่อนไหวของพีทว่ามีชิ้นงานกำลังทำอยู่ (`Doing`)
2. **คำนวณและประเมินเดดไลน์ (`test_agent_progress_deadline_packing`):** ดึงข้อมูล "ระบบบรรจุ" ของทีมพี่ไก่ (จิราภา) มาเฉลี่ยได้ 75% และตรวจสอบความพร้อมในการส่งมอบสำเร็จถูกต้อง
3. **กรองงานของบุคคลเฉพาะระบบ (`test_agent_specific_task_q`):** คัดกรองงานย่อยในระบบบรรจุที่ คิว กำลังพัฒนา (`Doing`)
4. **หาคนว่างงานข้ามทีม (`test_agent_idle_check`):** ดึงรายชื่อคนที่ไม่มีงาน Doing เลยในระบบออกมา ได้แก่ บิ๊ก (Done หมด), ตุ๊ก (Paused จัดซื้อ) และ วี (Done/Backlog)
5. **วิเคราะห์คอขวดสะสม (`test_agent_bottleneck_summary`):** แจ้งเตือนโครงการติดค้างที่มีสถานะ Paused และ Backlog พร้อมระบุหมายเหตุความล่าช้า

## 🎓 3. การประเมินผลและการส่งผลงาน (Capstone Submission Checklist)

โครงการนี้เตรียมความพร้อมในการส่งผลงานประกวดครบทุกหัวข้อประเมินของ Kaggle และ Google ดังนี้:

1. **ระบบมัลติเอเจนต์ (ADK):** ทำงานในระดับลึกข้าม 7 ชีตและแบ่งขั้นตอนคิดผ่าน State Orchestration ใน `app/agent.py`
2. **เซิร์ฟเวอร์ MCP:** ถูกพัฒนาลงตัวพร้อม Google Sheets API integration และ mock fallback
3. **ทักษะเอเจนต์ (Skills):** จัดทำทักษะอย่างเป็นรูปธรรมในโฟลเดอร์ทักษะ `.agents/skills/`
4. **ความปลอดภัย:** โมดูล PII masking กรองอีเมล/เบอร์โทรศัพย์ พร้อมรันเทส `test_security.py` ผ่านฉลุย
5. **ความสามารถในการปรับใช้งาน:** แดชบอร์ดStreamlit สวยงามพร้อมเปิดใช้งานและติดตั้งผ่าน requirements
6. **เครื่องมือ Antigravity:** **[คำแนะนำ]** ให้ทำการเปิดหน้าจอ IDE ที่บันทึกหน้าต่างแชตของ Antigravity AI Assistant นี้ไว้ในสไลด์/วิดีโอแนะนำตัว 5 นาที เพื่อเคลมคะแนนความร่วมมือในการออกแบบโค้ดและชุดทดสอบร่วมกับเอเจนต์
